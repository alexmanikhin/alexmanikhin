Explain how to create a personal web page in GitHub.

Setup repository
  1. Sign in to your GitHub account or create one if you haven't already.
  2. Create a new public repository named <username>.github.io with 'Public' option so that your page is accessible to everyone. Critical tip: If the first part of the repository doesn’t exactly match your username, it won’t work, so make sure to get it right.
  3. Click 'Create repository.
